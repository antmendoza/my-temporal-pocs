## Install dependencies

```bash
cd ctxpropagation
go mod tidy  
```


```bash
cd src_reproduction
npm install
```


## Run the go worker, that host the workflow
```bash
cd ctxpropagation
go run worker/main.go
```



## Run the ts worker, that host the activity
```bash
cd src_reproduction
npm run start.watch
```



## Start the workflow from go client
```bash
cd ctxpropagation
go run starter/main.go
```
